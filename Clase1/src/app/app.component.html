Aprendiendo Angular 2!

Primer tipo de bindeo: Trae una variable desde el component.ts hacia component.html

Variable "title":

{{title}}

Bindeo de doble vía. El input se bindea a la variable title de component.ts



Otro tipo de bindeo: trae una funcion desde TS hacia HTML


También se puede bindear un atributo de un objecto. El objeto es un atributo de la clase component.ts. Ejemplo:

Nombre del atributo "usuario":

{{usuario.nombre}}

Y una vez más, hacemos el doble bindeo. Esta vez bindeamos el atributo del objeto
